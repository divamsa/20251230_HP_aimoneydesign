# マーケティング改善提案書
## 株式会社マネーデザイン 生成AI導入支援サービス HP

**作成日**: 2025年1月
**レビュアー**: Webサイトマーケティングコンサルタント

---

## 🔴 緊急度：高（即座に改善すべき項目）

### 1. **ヒーローセクションの訴求力不足**

**現状の問題点**:
- 「FP✖️不動産✖️生成AIを謳う」という説明が、生成AI導入支援サービスとの関連性が不明確
- ユーザーの課題（痛み）が全く表現されていない
- ベネフィット（得られる価値）が抽象的すぎる
- CTAボタンが2つあり、選択肢が多すぎて迷う

**改善提案**:
```
【改善案1】ヒーローコピー
現状: 「生成AI導入支援サービス」
改善: 「生成AI導入で業務効率50%向上。100社以上の実績で選ばれる導入支援」

【改善案2】サブタイトル
現状: 「FP✖️不動産✖️生成AIを謳う株式会社マネーデザインの生成AI導入支援サービスです」
改善: 「中小企業・自治体・文化施設の業務改革を、実績豊富な専門チームが完全サポート」

【改善案3】CTA統一
現状: 「お問い合わせ」「サービス詳細を見る」の2つ
改善: 「無料相談を申し込む」（1つに統一、より具体的な行動を促す）
```

### 2. **実績数値の信頼性が低い**

**現状の問題点**:
- 「100+」「50+」「30+」という表現が曖昧
- 具体的な企業名・団体名が一切ない
- 実績の詳細（何を導入したか、どのような成果が出たか）が不明
- 「顧客満足度95%」の根拠が不明

**改善提案**:
```
【改善案1】数値の具体化
現状: 「100+ 導入企業数」
改善: 「導入企業数 120社以上（2024年12月時点）」

【改善案2】実績の可視化
- ロゴ掲載（許可が取れている企業のみ）
- 業種別の内訳を表示
- 導入規模別の内訳を表示

【改善案3】顧客満足度の根拠
- アンケート実施期間・回答数を明記
- 「95%の顧客が『満足』と回答（2024年実施、回答数80社）」
```

### 3. **サービス紹介が差別化できていない**

**現状の問題点**:
- 3つのサービス（コンサル、研修、開発）の違いが不明確
- 競合他社と何が違うのかが全く分からない
- 価格情報が一切ない（見積もりが必要なのは分かるが、相場感が分からない）
- サービス提供期間・納期の目安がない

**改善提案**:
```
【改善案1】差別化ポイントの明記
- 「FP・不動産の専門知識 × AI技術」という独自性を強調
- 「中小企業・自治体特化型の導入支援」を明確に
- 「導入後の運用サポートまで一貫対応」を強調

【改善案2】価格の目安
- 「初回相談：無料」
- 「導入支援：50万円〜（規模により変動）」
- 「研修・セミナー：1回5万円〜」
- ※完全な見積もりは無料相談後に提示

【改善案3】提供期間の目安
- 「コンサルティング：1〜3ヶ月」
- 「研修・セミナー：1日〜1週間」
- 「システム開発：3〜6ヶ月」
```

### 4. **導入事例が抽象的すぎる**

**現状の問題点**:
- 「A市役所」「B美術館」「C製造業」という仮名で、信頼性が低い
- 成果の数値（「50%削減」）の根拠が不明
- 導入前後の比較ができない
- 顧客の声（お客様の声）がない

**改善提案**:
```
【改善案1】事例の具体化
- 実際の企業名・団体名を掲載（許可が取れている場合）
- 導入前の課題を具体的に記載
- 導入後の成果を数値で明確に
- 導入期間・費用の目安を記載

【改善案2】お客様の声の追加
- 導入企業の担当者からのコメント
- 写真（可能であれば）
- 企業名・役職名・氏名（許可が取れている場合）

【改善案3】導入前後の比較
- 業務時間の削減：導入前 vs 導入後
- コスト削減：導入前 vs 導入後
- 顧客満足度：導入前 vs 導入後
```

### 5. **CTA（行動喚起）の配置と効果が弱い**

**現状の問題点**:
- トップページのCTAが最後に1つしかない
- 各セクションにCTAがない（ユーザーが興味を持った瞬間に行動を促せない）
- CTAのコピーが弱い（「お問い合わせ」→「無料相談を申し込む」の方が強い）
- 電話番号が目立つ場所にない

**改善提案**:
```
【改善案1】CTAの配置
- ヒーローセクション：1つ（メインCTA）
- 実績数値セクション後：1つ（「無料相談を申し込む」）
- サービス紹介セクション後：1つ（「サービス詳細を見る」）
- 導入事例セクション後：1つ（「導入事例をもっと見る」）
- ページ最下部：1つ（「今すぐ無料相談」）

【改善案2】CTAのコピー強化
現状: 「お問い合わせ」
改善: 「無料相談を申し込む（30秒で完了）」
改善: 「今すぐ無料相談（初回60分無料）」

【改善案3】電話番号の表示
- ヘッダーに固定表示（スマホではタップで発信）
- 「0120-XXX-XXX（平日9:00-18:00）」を目立つ場所に
```

## 🟡 緊急度：中（できるだけ早く改善すべき項目）

### 6. **会社情報ページの信頼性が低い**

**現状の問題点**:
- 会社情報ページの内容が不明（確認が必要）
- 代表者名・設立年・資本金などの基本情報が不明
- 会社の強み・特徴が不明
- 実績・受賞歴・認証などが不明

**改善提案**:
- 会社概要の充実（代表者、設立年、資本金、従業員数など）
- 会社の強み・特徴を明確に
- 実績・受賞歴・認証の掲載
- 会社のビジョン・ミッションの明記

### 7. **ブログコンテンツの戦略性が不明**

**現状の問題点**:
- ブログの目的が不明確（SEO対策？顧客教育？）
- カテゴリー構成が不明
- 更新頻度が不明
- コンテンツの質が不明

**改善提案**:
- ブログの目的を明確化（「生成AI導入のノウハウを共有」など）
- カテゴリー構成の最適化（「導入事例」「導入ノウハウ」「業界別情報」など）
- 更新頻度の明記（「週1回更新」など）
- SEO対策記事の充実

### 8. **お問い合わせフォームの最適化不足**

**現状の問題点**:
- 3つのフォームが縦に並んでいて、選択に迷う
- フォームが長すぎて離脱のリスクがある
- 必須項目が多すぎる可能性がある
- 送信後の次のステップが不明確

**改善提案**:
- フォームのタブ化（「一般問い合わせ」「資料請求」「無料相談」）
- 必須項目の最小化（名前、メール、電話、問い合わせ内容のみ）
- 送信後の自動返信メールの内容を充実
- 送信後の次のステップを明記（「24時間以内にご返信いたします」など）

### 9. **SEO対策の強化が必要**

**現状の問題点**:
- メタディスクリプションが各ページで統一されていない可能性
- 見出しタグ（H1, H2, H3）の構造が不明
- 内部リンク構造が不明
- 外部リンク（被リンク）戦略が不明

**改善提案**:
- メタディスクリプションの最適化（各ページでキーワードを含める）
- 見出しタグの構造化（H1は1ページに1つ、H2-H6を適切に使用）
- 内部リンクの充実（関連ページへのリンクを適切に配置）
- 外部リンク戦略の策定（業界メディアへの寄稿、プレスリリースなど）

### 10. **モバイルUXの最適化**

**現状の問題点**:
- モバイルでの表示確認が必要
- タッチターゲットのサイズが適切か不明
- モバイルでの読み込み速度が不明
- モバイルでのCTAの配置が不明

**改善提案**:
- モバイルでの表示確認と最適化
- タッチターゲットのサイズを48px×48px以上に
- 画像の最適化（WebP形式、遅延読み込み）
- モバイルでのCTAの配置を最適化（画面下部に固定など）

## 📝 まとめ

このサイトは技術的には問題ありませんが、**マーケティングの視点から見ると改善の余地が大きい**です。

特に重要なのは：
1. **訴求力の強化**（ヒーローセクション、サービス紹介）
2. **信頼性の向上**（実績の具体化、導入事例の充実）
3. **行動喚起の最適化**（CTAの配置とコピー）
4. **差別化の明確化**（競合との違いを明確に）

これらの改善により、コンバージョン率の向上が期待できます。
